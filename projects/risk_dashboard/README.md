# Risk Dashboard

Interactive Streamlit dashboard for portfolio risk analysis and factor decomposition.

## What it does

- **Risk Overview**: Sharpe, Sortino, and Calmar ratios; VaR and CVaR (historical and parametric); drawdown analysis with interactive charts
- **Factor Exposure**: CAPM regression (alpha, beta, R-squared, t-statistics); Fama-French 3-factor loadings; rolling beta visualization
- **Comparison**: Side-by-side ticker comparison; cumulative returns overlay; correlation heatmap

## How to run

```bash
pip install -r requirements.txt
streamlit run app.py
```
